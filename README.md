# 최소 재현 레포

## 플젝 세팅

### 의존성 설치

```
corepack enable
pnpm i
```

### 환경변수

`.env` 와 `.dev.vars` 를 루트에 생성해주세요.  
그리고 아래 값으로 채워주세요.

https://www.notion.so/yourssu/env-1c76915d697880a9bb77c72cbb9b6a3a?pvs=4

### 개발 서버 실행

```
pnpm dev
```

## 배포

```
pnpm run deploy
```
